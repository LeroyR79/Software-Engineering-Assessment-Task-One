# Software-Engineering-Assessment-Task-One
(What the code is and what it does)
## Necessary installations
- User needs to install requirements through terminal with the command - 'install -r requirements.txt'. This will install all necessary dependencies and give access to the requirements.txt file.
- The user then needs to make sure that they run the code from the main.py file, as it is the only one that can run the API.
## Necessary files and important features
- There are 8 files in this folder (including .gitignore and api_key.py which should be hidden from the regular user). The functions.py and the main.py folders include vital code needed to initiate the code and get information from the API.
